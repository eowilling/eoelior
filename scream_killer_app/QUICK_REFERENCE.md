# 🚀 ScreamKiller 快速參考卡

## ⚡ 快速啟動
```batch
# 雙擊執行
開始執行.cmd
```

## 📝 使用流程

### 1️⃣ 上傳影片
- 格式：MP4 / MOV
- 大小：< 500MB
- 長度：< 10 分鐘

### 2️⃣ 選擇模式

#### 🎚️ 手動模式（簡單）
拖動滑桿 → 點擊處理

#### 🤖 智能模式（推薦）
1. 設定**參考片段**（必填）
   - 格式：`MM:SS` 例如 `01:30`
   - 建議：選歌手獨唱段落
   - 長度：5-15 秒
   - 數量：1-3 個

2. 設定**抑制區**（選填）
   - 標記尖叫特別大聲的時段
   - 可不設定

### 3️⃣ 處理下載
點擊 🚀 開始處理 → 等待完成 → 下載影片

## 🎯 實用技巧

### ✅ DO（建議）
- ✅ 選擇清晰的歌手獨唱作為參考
- ✅ 多設幾個參考片段提高準確度
- ✅ 只標記最嚴重的尖叫段落
- ✅ 保留原始影片備份

### ❌ DON'T（避免）
- ❌ 不要選和音/合唱當參考
- ❌ 不要把整首歌標為抑制區
- ❌ 處理時不要關閉瀏覽器
- ❌ 不要同時處理多個影片

## 🐛 常見問題

### Q1: 結果不理想？
**A:** 調整參考片段，選更清晰的獨唱段落

### Q2: 處理很慢？
**A:** 正常現象，AI 需要時間。可嘗試較短影片

### Q3: 出現錯誤？
**A:** 檢查：
- Python 版本 >= 3.9
- 磁碟空間充足
- 記憶體 >= 8GB

### Q4: 畫質會下降嗎？
**A:** 不會！視頻不重新編碼，保持原畫質
