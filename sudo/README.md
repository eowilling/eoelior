# 数独游戏 - 无限提示版
一款功能完整的数独游戏，支持五星级难度选择、无限制提示、存档功能，且没有时间限制。
## 功能特性
### 🎯 核心功能
- **五星级难度系统**：从1星（简单）到5星（大师级）的难度选择
- **无限制提示**：随时可以获取提示，帮助您解决难题
- **存档/读档**：支持保存和加载游戏进度
- **无时间限制**：您可以慢慢思考，没有时间压力
### 🎮 游戏界面
- **直观的图形界面**：使用tkinter创建的美观界面
- **3x3宫格视觉分隔**：清晰的数独网格显示
- **颜色区分**：初始数字和用户填入数字用不同颜色显示
- **状态提示**：实时显示游戏状态和操作提示
### 🎲 游戏规则
- 标准9x9数独规则
- 每行、每列、每个3x3宫格内数字1-9不重复
- 初始数字（蓝色）不可修改
- 用户填入数字（黑色）可以修改或清除
## 安装和运行
### 系统要求
- Python 3.6或更高版本
- tkinter（Python标准库，通常已预装）
### 运行方法
1. 确保已安装Python
2. 下载所有游戏文件到同一目录
3. 运行主程序：
   ```bash
   python main.py
   ```
## 操作说明
### 基本操作
1. **选择难度**：点击界面顶部的难度星级（1-5星）
2. **开始新游戏**：点击"新游戏"按钮
3. **填入数字**：
   - 点击空白单元格选中（会变成黄色）
   - 点击下方的数字按钮（1-9）填入数字
4. **清除数字**：选中单元格后点击"清除"按钮
5. **获取提示**：点击"提示"按钮，系统会自动填入一个正确答案
6. **保存游戏**：点击"保存"按钮，选择保存位置
7. **加载游戏**：点击"加载"按钮，选择之前保存的游戏文件
### 界面说明
- **蓝色数字**：初始数字，不可修改
- **黑色数字**：用户填入的数字
- **黄色背景**：当前选中的单元格
- **灰色背景**：初始数字的单元格
- **状态栏**：显示当前操作状态和提示信息

## 技术特点
### 核心算法
- **回溯算法**：用于生成完整数独解和求解
- **唯一解验证**：确保生成的谜题有且仅有一个解
- **智能数字移除**：根据难度级别智能移除数字
### 数据结构
- **二维数组**：存储数独棋盘状态
- **JSON格式**：用于游戏存档，便于阅读和编辑
### 代码结构
- **sudoku_core.py**：核心游戏逻辑
- **sudoku_gui.py**：图形用户界面
- **main.py**：主程序入口

## 常见问题
### Q: 游戏无法启动怎么办？
A: 请确保已安装Python 3.6或更高版本，并且tkinter库已正确安装。
### Q: 如何知道我填入的数字是否正确？
A: 游戏会实时验证数字的有效性，如果数字违反数独规则，将无法填入。
### Q: 提示功能有使用次数限制吗？
A: 没有！您可以无限次使用提示功能。
### Q: 保存的游戏文件在哪里？
A: 默认保存在您选择的位置，建议创建专门的文件夹来管理游戏存档。
### Q: 可以修改初始数字吗？
A: 不可以，初始数字（蓝色显示）是固定的，只能修改您自己填入的数字。
