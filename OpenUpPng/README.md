# OpenUpPng - 圖片轉 PNG 上傳工具

一個現代化的圖片上傳工具，自動將上傳的圖片轉換為 PNG 格式並提供公開分享連結。

## 功能特色

- 🖼️ **自動轉換**: 支援 JPG, JPEG, GIF, BMP, WEBP 格式，自動轉換為 PNG
- 📤 **拖曳上傳**: 支援拖曳檔案或點擊選擇
- 🔗 **公開連結**: 上傳後立即獲得可分享的公開 URL
- 📋 **一鍵複製**: 快速複製圖片連結到剪貼簿
- 📸 **圖片庫**: 查看最近上傳的所有圖片
- ⏰ **自動清理**: 圖片在上傳後 3 天自動刪除
- 🎨 **現代設計**: 精美的深色主題界面，支援響應式設計

## 系統需求

- PHP 7.4 或更高版本
- GD 圖片處理庫
- Apache 或 Nginx 網頁伺服器

## 安裝步驟

1. 將所有檔案上傳到您的網頁伺服器
2. 確保 `uploads/` 目錄有寫入權限：
   ```bash
   chmod 755 uploads/
   ```
3. 在瀏覽器中訪問 `index.html`

## 使用方式

1. **上傳圖片**:
   - 拖曳圖片到上傳區域，或
   - 點擊上傳區域選擇檔案

2. **複製連結**:
   - 上傳成功後，點擊「複製」按鈕
   - 連結已複製到剪貼簿，可以分享給其他人

3. **查看圖片庫**:
   - 在下方圖片庫查看所有已上傳的圖片
   - 每張圖片都顯示剩餘有效時間

## 技術規格

- **最大檔案大小**: 10MB
- **支援格式**: JPG, JPEG, PNG, GIF, BMP, WEBP
- **輸出格式**: PNG
- **保存期限**: 3 天自動刪除

## 自動清理機制

系統會在以下時機自動清理超過 3 天的圖片：
- 每次上傳新圖片時
- 每次載入圖片庫時

您也可以設定 cron job 定期執行清理：
```bash
# 每小時執行一次清理
0 * * * * php /path/to/OpenUpPng/cleanup.php
```

## 安全性建議

1. 限制上傳檔案大小
2. 驗證檔案類型
3. 使用唯一檔名防止覆蓋
4. 定期清理舊檔案
5. 考慮添加使用者驗證（如需要）
