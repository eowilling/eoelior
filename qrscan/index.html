快速啟動指南

1

確認運行環境

本工具基於 Python 開發，請確保您的系統已安裝 Python 3.8 或以上版本。

下載 Python
2

安裝依賴庫

初次使用請執行以下指令安裝所需模組（OpenCV, Pillow 等）。

pip install -r requirements.txt
3

啟動圖形介面

執行主程式以開啟 Enhanced GUI 介面。

python qr_enhancer_tool_enhanced.py
小撇步 (Windows): 您也可以直接雙擊工具目錄下的 啟動工具.bat 腳本，透過簡易選單進行操作。
