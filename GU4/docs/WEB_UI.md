# 台股智能分析系統 - Web UI 使用指南

## 🚀 快速啟動

### 1. 啟動 Web 伺服器
```bash
python web_app.py
```

### 2. 開啟瀏覽器
訪問：http://localhost:5000

### 3. 開始使用
- 在左側控制面板設定分析參數
- 點擊「開始分析」執行分析
- 在右側查看即時結果

---

## 📋 功能說明

### 選股模式

#### 🔸 手動指定
直接輸入股票代碼，用逗號分隔
```
例如: 2330,2454,0050
```

#### 🔸 智能選股
系統自動選擇市場熱門股票

**選股方法：**
- **三大法人買超** - 外資、投信、自營商買超排行
- **漲幅排行** - 當日漲幅最大的股票
- **成交量排行** - 成交量最大的股票
- **綜合評分** - 綜合上述三種方法

**選股數量：** 1-20 支股票

---

## 🎛️ 進階設定

### 分析延遲
設定每支股票分析之間的延遲時間（秒），避免 API 限流
- 建議值：3 秒
- 範圍：1-10 秒

### 儲存設定
將當前設定儲存到 `.env` 檔案

## ⚠️ 注意事項

1. **單次分析** - 同時只能執行一個分析任務
2. **網路連線** - 智能選股需要網路連線
3. **API 配額** - 注意 Gemini API 的每日配額限制
4. **瀏覽器** - 建議使用 Chrome、Edge、Firefox 最新版本

## 🆘 常見問題

### Q: 如何停止正在執行的分析？
A: 重新整理頁面即可停止

### Q: 分析失敗怎麼辦？
A: 檢查 `.env` 配置，確保 API 金鑰正確

### Q: 可以同時開啟多個瀏覽器分頁嗎？
A: 可以，但只有一個分析任務會執行

### Q: 結果可以匯出嗎？
A: 目前版本暫不支援，可以複製文字內容

## 🔐 安全性

- Web UI 預設只監聽本機（localhost）
- 如需允許區域網路訪問，修改 `web_app.py` 中的 `host` 參數
- 不要將含有 API 金鑰的系統暴露到公網

---

## 🚀 進階使用

### 修改監聽埠
編輯 `web_app.py`：
```python
app.run(host='0.0.0.0', port=8080)
```

### 允許區域網路訪問
```python
app.run(host='0.0.0.0', port=5000)
```
然後在其他裝置訪問：`http://您的電腦IP:5000`
