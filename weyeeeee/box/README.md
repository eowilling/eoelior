# 尾牙抽獎系統 (Lucky Draw Box)
一個簡單易用的尾牙抽獎工具，具有科技感界面和流暢動畫效果。
## 功能特色
- 🎰 **抽獎功能**：選擇獎項、滾動動畫、中獎特效
- 🏆 **獎項管理**：自定義獎項、設置數量、等級管理
- 👥 **人員管理**：批量導入、單個添加、狀態追蹤
- 📊 **結果管理**：實時顯示、導出CSV、歷史記錄
- 🎨 **科技界面**：深色主題、響應式設計、動畫效果
## 使用方法
1. 在 `index.html` 中設置您的獎項和人員
2. 用瀏覽器打開 `index.html` 即可使用
3. 數據自動保存在瀏覽器本地存儲中
## 技術規格
- 純前端實現（HTML + CSS + JavaScript）
- 使用 localStorage 本地存儲
- 無需服務器，可直接部署到 GitHub Pages
## 部署到 GitHub Pages
1. 將此專案上傳到 GitHub 倉庫
2. 在倉庫設定中啟用 GitHub Pages
3. 選擇 `main` 分支作為源
