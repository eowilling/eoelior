# 尾牙抽獎系統 - 使用說明
## 🎉 系統特色
- **科技感界面**：深色主題搭配霓虹光效，營造高端大氣的視覺體驗
- **動畫效果**：流暢的粒子背景動畫和中獎特效，提升參與感
- **簡單易用**：直觀的操作界面，無需專業技能即可上手
- **數據持久化**：本地存儲功能，確保數據不丟失
- **響應式設計**：支持各種屏幕尺寸，適配大屏展示
## 📋 功能介紹
### 1. 獎項設置
- 支持自定義獎項名稱和數量
- 可添加多個不同等級的獎項
- 自動追蹤獎品剩餘數量
### 2. 參與人設置
- 支持批量導入參與人名單
- 每行輸入一個姓名即可
- 自動標記已中獎人員
### 3. 抽獎功能
- 單次抽獎模式
- 滾動動畫效果
- 自動跳過已中獎人員
- 即時顯示中獎結果
### 4. 歷史記錄
- 完整的中獎記錄查詢
- 支持導出為CSV格式
- 可清除歷史記錄
## 🚀 使用步驟
### 第一步：設置獎項
1. 點擊「獎項設置」選項卡
2. 輸入獎項名稱和數量
3. 點擊「添加獎項」增加更多獎項
4. 點擊「保存設置」完成配置
### 第二步：添加參與人
1. 點擊「參與人設置」選項卡
2. 在文本框中輸入參與人名單（每行一個姓名）
3. 點擊「添加參與人」完成導入
### 第三步：開始抽獎
1. 點擊「抽獎」選項卡
2. 點擊「開始抽獎」按鈕
3. 等待滾動動畫
4. 點擊「停止抽獎」確定中獎者
5. 系統自動顯示中獎結果
### 第四步：查看記錄
1. 點擊「歷史記錄」選項卡
2. 查看所有中獎記錄
3. 可導出記錄為CSV文件
## ⚙️ 系統要求
- 現代瀏覽器（Chrome、Firefox、Edge、Safari）
- 支持JavaScript
- 建議使用1920x1080或更高分辨率的大屏展示
## 🔧 技術特點
- **純HTML/CSS/JavaScript**：無需後端服務器
- **本地存儲**：使用localStorage保存數據
- **響應式設計**：自動適配不同屏幕尺寸
- **動畫優化**：使用CSS3動畫提升性能
## 📝 注意事項
1. **數據備份**：建議定期導出中獎記錄作為備份
2. **瀏覽器兼容**：請使用最新版本的瀏覽器以獲得最佳體驗
3. **屏幕設置**：大屏展示時建議使用全屏模式
4. **網絡連接**：雖然系統可離線使用，但首次加載需要網絡連接
## 🎯 最佳實踐
1. **活動前準備**：
   - 提前設置好所有獎項
   - 導入完整的參與人名單
   - 測試抽獎流程
2. **活動中操作**：
   - 使用大屏展示效果更佳
   - 建議由專人負責操作
   - 及時導出中獎記錄
3. **活動後整理**：
   - 導出完整的中獎記錄
   - 保存數據作為檔案
   - 可重置系統供下次使用
## 🆘 常見問題
**Q: 如何重置系統？**  
A: 在抽獎頁面點擊「重置系統」按鈕，確認後將清除所有中獎記錄。
**Q: 數據會丟失嗎？**  
A: 系統使用本地存儲，只要不清除瀏覽器緩存，數據會一直保存。
**Q: 支持多少人參與？**  
A: 理論上沒有限制，但建議不超過1000人以保證性能。
**Q: 可以修改已設置的獎項嗎？**  
A: 可以，在獎項設置頁面直接修改並保存即可。
