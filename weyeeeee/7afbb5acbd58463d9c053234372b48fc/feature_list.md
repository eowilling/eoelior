# 尾牙抽奖系统核心功能需求列表
## 一、奖项管理模块
1. 奖项等级配置
   - 支持设置特等奖/一等奖/二等奖/三等奖/参与奖等多级奖项
   - 每级奖项可自定义名称、图标、描述文案
2. 奖项数量控制
   - 设置各等级奖项总数量（如特等奖1名、一等奖5名）
   - 支持“数量不限”模式（适用于阳光普照奖）
3. 中奖概率配置
   - 按奖项等级设置基础概率（如特等奖0.5%、一等奖3%）
   - 支持“保底机制”（如未中高等级奖项时自动参与低等级抽奖）
4. 奖项状态管理
   - 启用/禁用特定奖项（用于临时调整活动规则）
   - 奖项库存预警（剩余数量低于阈值时提醒管理员）
## 二、参与人管理模块
1. 名单导入功能
   - 支持Excel/CSV格式批量导入（含姓名、部门、工号字段）
   - 提供模板下载功能（规范导入格式）
2. 人员信息维护
   - 单条新增/编辑/删除参与人信息
   - 部门树形结构展示（支持按部门筛选参与人）
3. 人员分类管理
   - 自定义标签体系（如“管理层”“新员工”“优秀员工”）
   - 按标签分组抽奖（如单独抽取部门奖项）
4. 参与状态控制
   - 标记“不参与抽奖”人员（如已离职员工）
   - 记录参与人签到状态（关联活动现场签到）
## 三、抽奖流程模块
1. 抽奖模式选择
   - 单次抽奖（逐一点击抽取单个奖项）
   - 批量抽奖（一次性抽取同等级多个奖项）
2. 智能过滤机制
   - 自动跳过已中奖人员（支持“允许重复中奖”开关）
   - 黑名单过滤（排除不符合条件人员）
3. 抽奖过程控制
   - 手动开始/暂停/重置抽奖流程
   - 设置抽奖倒计时（增强现场紧张感）
4. 特殊场景支持
   - 补抽功能（针对弃奖/无效奖重新抽取）
   - 试抽模式（不记录结果的压力测试）
## 四、结果展示模块
1. 实时展示功能
   - 大屏幕动态显示中奖者姓名、奖项、部门
   - 中奖者照片轮播（需提前上传照片）
2. 历史记录查询
   - 按时间/奖项等级/部门多维度筛选记录
   - 中奖者领奖状态标记（未领/已领/过期）
3. 结果导出分享
   - 生成中奖名单PDF/Excel（含公司LOGO）
   - 社交媒体分享模板（一键生成喜报）
4. 数据分析视图
   - 各部门中奖分布统计图表
   - 奖项抽取耗时趋势分析
## 五、界面交互模块
1. 科技感UI设计
   - 深色主题搭配霓虹光效（蓝紫渐变为主色调）
   - 玻璃拟态控件（半透明磨砂质感按钮/面板）
2. 动画效果实现
   - 抽奖转盘/滚动名单动画（速度可调）
   - 中奖烟花/礼花粒子特效
3. 响应式布局
   - 适配4K大屏/投影仪（16:9比例优化）
   - 移动端H5适配（管理员手机控场）
4. 无障碍设计
   - 键盘快捷键支持（空格开始/ESC重置）
   - 屏幕阅读器兼容（关键信息语音播报）
