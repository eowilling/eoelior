手動修復指南

如果無法使用 Python 工具，可以按照以下步驟手動優化：

階段一：快速診斷（5分鐘）

1. 打開日志目錄

explorer %LocalAppData%\Files

或者按 Win+R 輸入上述命令

2. 檢查日志文件

  • debug.log - 主應用日志
  • debug_server.log - 後台服務日志

階段二：基礎優化（10分鐘）

注意：80% 的問題可通過此階段解決！

調整界面設置
  • 打開 Files 設置（Ctrl+,）
  • 前往「外觀」選項卡
  • 圖標大小：選擇「小圖標」
  • 禁用「顯示縮略圖」
優化性能配置

編輯配置文件：%LocalAppData%\Files\settings.json

{
  "Performance": {
    "MaxVisibleItems": 500,
    "EnableVirtualization": true
  },
  "PdfPreviewMaxPages": 20,
  "MaxPreviewSizeMB": 10
}

階段三：常見問題修復

啟動崩潰

刪除會話緩存：

del "%LocalAppData%\Files\last_session.json"

界面卡頓

降低顯示項目數量至 300-500

避免大文件夾使用縮略圖模式

PDF 預覽崩潰

更新 WebView2：

winget install Microsoft.EdgeWebView2Runtime

拖放失敗

取消「以管理員身份運行」

檢查權限隔離設置
